# Multimodal Study Assistant 📚

Віртуальний асистент для навчання на базі Google Gemini 2.5 Flash. Застосунок дозволяє завантажити фото конспекту або скриншот тексту, розпізнає головні тези, пояснює складні терміни та генерує інтерактивний квіз для перевірки знань.

## Технології
- Python 3
- Streamlit (UI)
- Google GenAI SDK (Gemini API)
- Pydantic (Structured Output)
- uv (Package manager)
- Pytest (Тестування)

## Встановлення та запуск

1. Переконайтеся, що у вас встановлено пакетний менеджер `uv`.
2. Клонуйте репозиторій та перейдіть у папку проєкту.
3. Встановіть залежності:
   ```bash
   uv sync
   ```
4. Створіть файл `.env` у корені проєкту та додайте туди свій API-ключ:
   ```env
   GEMINI_API_KEY="ваш_ключ_тут"
   ```
5. Запустіть застосунок:
   ```bash
   uv run streamlit run src/assistant/main.py
   ```

## Запуск тестів
Щоб перевірити коректність роботи логіки, запустіть тести:
```bash
uv run pytest
```
